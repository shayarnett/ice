# Getting Started

## Installation

```bash
bun add ice-ssg
```

## Create Your First Page

Create a Markdown file with frontmatter:

```markdown
---
title: My Page
layout: default
---

# Hello World
```

## Run the Dev Server

```bash
bun run dev
```

Your site will be available at `http://localhost:4000`.
