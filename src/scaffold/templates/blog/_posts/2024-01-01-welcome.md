Welcome to your new blog powered by **Ice**, a fast static site generator built on Bun.

## Getting Started

Edit this post or create new ones in the `_posts/` directory. Posts use the naming convention `YYYY-MM-DD-slug.md`.

### Features

- Markdown with frontmatter
- Liquid templates
- RSS and JSON feeds
- Live reload during development
- Fast incremental builds

## What's Next?

- Customize `_layouts/default.liquid` to change the site design
- Edit `assets/css/main.css` for styling
- Add more posts in `_posts/`
- Deploy to GitHub Pages with the included workflow

Happy writing!
